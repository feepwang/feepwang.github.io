C++的指针

Posted on Feb 12, 2021

C++普通指针

下面记录几个常用的C++普通指针的初始化：

  1. int *ptr1;
   
int i = 10;
int *ptr2 = &i;

  2. int *ptr3 = new int;
int *ptr4 = new int(100);

const、指针

const修饰符用来表示一个类型的常量。

  • const的一般用法：
//	常量i
const int const_i;
//	常量引用
const int &const_reference_i;
  • const与指针
//	常量指针（指向常量的指针）
const int * ptr_to_const;
//	指针常量
int *const const_ptr;
//	指向常量的指针常量
const int *const const_ptr_to_const;

智能指针

C++11引入了几类新的指针：shared_ptr、unique_ptr、weak_ptr。

  • 对比普通指针的new、delete命令，智能指针能自动管理内存空间的释放。

  • 智能指针可以看作一个系统内置的类模版。

  • 这个类模版里面存一个给定类型的指针，重载了*、->运算符以模拟指针的操作。

shared_ptr

shared_ptr能够共享管理的指针，直到内置的计数器归零才释放内置指针指向的对象的内存空间。

shared_ptr常用的初始化：

std::shared_ptr<int> sptr1 = std::make_shared<int>();
std::shared_ptr<int> sptr2(new int);
